Создать консольное приложение, способное производить сортировку массива структур. 
 
Входные данные​: 

набор параметров окружностей: 
- Координата X; 
- Координата Y; 
- Радиус окружности R. 

Количество вводимых окружностей произвольно и определяется пользователем во время работы. 
Выходные данные​:

список сохраненных окружностей в порядке 
невозрастания или неубывания по одной из координат или радиуса окружности формата: 
X=<x>; Y=<y>; R=<r> 
 
Пример: 
X=2.5; Y=7.2; R=5.1
X=2.7; Y=-1.0; R=2
X=2.7; Y=3.4; R=13
X=2.9; Y=0; R=7.4
 
● Пользователь должен иметь возможность сортировать все окружности по одному из трех
параметров по невозрастанию или неубыванию. 
● Пользователь должен иметь возможность ввести параметры окружностей один раз
(в рамках одного запущенного экземпляра программы) и в дальнейшем выбирать
порядок сортировки и параметры любое количество раз до закрытия программы. 
● У пользователя должна быть возможность вводить новые окружности и 
удалять весь массив без закрытия программы. Производить выбор 
функции (ввод, сортировка, сброс) с помощью ввода с клавиатуры. 
● При некорректном выборе возвращать в последнее меню с выводом сообщения о некорректном выборе. 

Вводимые параметры окружности должны храниться в формате 
структуры (struct). Массив для хранения каждого отдельного элемента,
соответствующего формату структуры, должен быть динамическим (указатели). 
Продумать и обезопасить весь ввод от некорректных данных — приложение 
должно оставаться работоспособным при любом некорректном вводе. 

В качестве результата работы прикрепить архив (.zip) с исходным кодом и отчет.
Отчет должен состоять из: 
- Описания логики программы (псевдокод, блок-схема или последовательное описание кода); 
- Руководства пользователя; 
- Примера работы программы. 

По желанию: 
- (Низкая сложность) Для упрощения проверки работоспособности программы сделать возможность
случайной генерации окружностей, т.е. параметров элементов массива. 
- (Средняя сложность) Вместо того, чтобы менять местами сами элементы массива, сделать
отдельный массив указателей на элементы массива и менять местами их, а потом выводить элементы
по адресам из массива указателей. 
- (Высокая сложность) Для экономии памяти, если пользователь ввел окружность с радиусом,
с которым в массиве уже существует другая окружность, записать координаты новой окружности
в тот же элемент структуры.
